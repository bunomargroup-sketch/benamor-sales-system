# Benamar Sales System

نظام بيع ومخزون منفصل لمجموعة بن عمر، مرتبط بنفس Supabase المستخدم في النظام الحالي.

## المرحلة الأولى

- الفروع والمخازن:
  - فرع 11 يونيو
  - فرع السراج
  - مخزن جنزور
- إدارة الموردين
- كشف حساب الموردين
- تسجيل دفعات الموردين
- تجهيز جداول الشراء والمخزون والتحويلات للمرحلة القادمة

## طريقة التركيب

1. افتح Supabase.
2. SQL Editor.
3. انسخ محتوى ملف `supabase-pos-setup.sql` وشغّله مرة واحدة.
4. ارفع الملفات التالية إلى GitHub repository:
   - `index.html`
   - `supabase-pos-setup.sql`
   - `README.md`
5. افتح الرابط:

```text
https://bunomargroup-sketch.github.io/benamar-sales-system/?v=1
```

## ملاحظات مهمة

- هذا النظام منفصل عن `pricechecker2` ولا يغيّر الموقع الحالي.
- يستخدم نفس Supabase project.
- الجداول الجديدة تبدأ بـ `pos_` لتفادي التعارض مع الجداول الحالية.
